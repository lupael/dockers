# Docker-Laravel v8

## Build
```
docker-compose build
```

## RUN
```
docker-compose up -d
```

## DOWN
```
docker-compose down
```

## PHP ARTISAN MIGRATE (DATABASE)
```
docker-compose exec php php artisan migrate
```

## CLI PHP
```
docker-compose exec php bash
```

## CLI MYSQL
```
docker-compose exec mysql bash
```

## CLI NGINX
```
docker-compose exec nginx bash
```
